**Expense Tracker**
A full-stack Expense Tracker application built using React, Spring Boot, and PostgreSQL. Users can register, log in, add expenses, view their expenses, and manage their spending efficiently
**Features**
* User Registration & Login
* Add Expenses
* View Expenses
* Delete Expenses
* PostgreSQL Database Integration
* Responsive UI
**Tech Stack**
**Frontend**
* React
* TypeScript
* Vite
* Axios
* Tailwind CSS
**Backend**
* Spring Boot
* Spring Data JPA
* Spring Security
* Maven
**Database**
* PostgreSQL (Neon)
**Deployment**
* Render
* GitHub
